# Assets (imágenes)

Carpeta de imágenes y recursos estáticos que usa el sitio.

## Carpetas principales

- `home/`: imágenes usadas en la portada.
- `products/`: imágenes de productos (usadas desde `src/data/products.json`).
- `img/`: gifs/recursos puntuales usados en páginas.
- `cousy/`: recursos varios de la marca (si aplica).

## Descargas temporales

Las carpetas que empiezan con `_` (por ejemplo `_cousy-download`, `_productos-download`, etc.) contienen originales/variantes descargadas.
Si ya no se necesitan, pueden limpiarse con el script `scripts/cleanup-temp-downloads.ps1`.
